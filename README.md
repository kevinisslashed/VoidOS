A lightweight desktop in your own browser, especially built for musicians, where they can write their song, pull the basics, and more!

#Features
Notes: you can write your lyrics here
Metronome: a BPM counter so that writing sessions become easier
Tuner: For basic pitch preference
Setlist: A list of your songs
Calculator: A basic calculator
Terminal: a tiny command line (help, date, note, BPM[n])
Calendar: Current month view
Custom wallpaper and continuous background music made by me

#Run it locally
1. Download this repo
2. Make sure all files are in one folder
3. Open index.html to run your WebOS

#Notes
1. Windows can be dragged by their title bar.
2. Background music loops automatically after your first click on the page.
